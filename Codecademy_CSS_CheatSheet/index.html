CSS Cheatsheet

HTML Components to Link to a Stylesheet

HTML Description
<link href="style.css" type="text/css" rel="stylesheet"> creates a link to an external custom style sheet
<link href="https://fonts.googleapis.com/css? family=Droid%20Serif" rel="stylesheet">> imports a font

CSS Properties and Attributes

Color

CSS Property: Value Use/Meaning
background-color: aqua; sets the background color (non-text) of the current container Quacklit CSS Colors
color: blue; sets the foreground color (text) of the current container Quacklit CSS Colors
opacity: 0.5; level of transparency on a scale of 0-1.0
color: rgb(255, 0, 127); red, green, and blue intensities on a scale of 0-255
color: rgba(255, 0, 127, 0.75); added alpha value is essentially opacity
color: #FF007F; hexadecimal representation of rgb coding
color: hsl(12, 100%, 50%); hue (0-360), saturation (0-100%), lightness (0-100%)
color: hsla(12, 100%, 100%, 0.8); added alpha value is essentially opacity

Text

CSS Property: Value Use/Meaning
font-family: 'Times New Roman', Times, serif; typeface style ranging from nicest to most likely available by default
@font-face {
font-family: 'Glegoo';
src: url('../fonts/Glegoo-Regular.ttf') format('truetype');
}		 imports a font
font-size: 16px; size of text
font-weight: 400; lightness/boldness on a scale from 100-900 in multiples of 100; or words like: normal, bold
font-style: italic describes slants and cursive styles; normal, oblique
line-height: 10px; describes text height plus space between lines
text-align: left; justification with respect to parent container; right, center, justify
text-transform: uppercase; convert text to uppercase, lowercase, or capitalized
text-decoration: underline; applies lines (or lack of) to text; none, overline, line-through
text-indent: 20px; indents first line of text in an element
letter-spacing: 2px; set horizontal space between characters
word-spacing: 0.3em; set horizontal space between words

Box

CSS Property: Value Use/Meaning
border: 3px solid coral; draw a border around the container with given: width, style, and color
border-radius: 5px; round the corners of a container
padding: 3px; space between container content and border
margin: 20px; space between container border and its true outer edge
background-image: url("nyan-cat.gif"); sets background image of container
overflow: scroll; determines how to handle content that is larger than its container.
box-sizing: content-box; padding and border are in addition to width/height (default)
box-sizing: border-box; padding and border are included in width/height (content shrinks)
z-index: -1; location into or out of the page, for overlapping; larger is closer
display: block; content wrapped tightly on top and bottom, but gets page width by default, size can be set
display: inline; boundaries wrap tightly to content without newlines or size control
display: inline-block; elements can appear adjacent with set dimensions
display: none element's visibility is completely removed from page
position: fixed; sticks container to viewport relative to the page
position: absolute; every other element ignores this one and position becomes relative to parent
position: relative; dictate position relative to default static location
top: 50px; with relative position, distance down from the top
left: 120px; with relative position, distance from the left
position: static; default, inline, block, and inline-block rules apply
position: sticky; element scrolls with page up to a sticky point
float: right; moves elements to the left or right within their container
clear: both; avoids blocking elements by clearing none, left, right, or both sides
visibility: hidden; removes element's visibility, but saves space
outline-style: dotted; set border outline style; dashed, solid, double, groove, ridge, inset, outset
outline-color: red; set border outline color
outline-width: 4px; set border outline width
outline: 5px ridge pink shorthand for width, style, and color
outline-offset: 15px; adds space between the border and outline

Pseudo-Elements

Pseudo-Elements Description
a:link {} sets the styles for links not yet selected
a:visited {} set styles for links that have been selected
a:hover {} set styles during a mouse over
a:focus {} set styles for item that has focus
a:active {} set styles during mouse click
p.intro:first-letter {} set styles for the first letter in an element
p.intro:first-line {} set styles for the first line of text in an element
